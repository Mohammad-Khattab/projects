Snake — Neon Cyberpunk 3D

Full screen-by-screen design overview. All screens. Three.js WebGL game canvas. Each map has its own neon theme.

Architecture

Renderer

Three.js r165 loaded from CDN. Game canvas replaced with a WebGL renderer. Snake segments = BoxGeometry with MeshStandardMaterial + emissive. Food = IcosahedronGeometry with a PointLight attached. Grid floor = PlaneGeometry with custom ShaderMaterial.

Camera

PerspectiveCamera slightly tilted (~12° pitch, -5° yaw) above the board — not flat top-down. Camera subtly shakes on food eaten. All gameplay logic unchanged — only the render layer changes.

UI screens

Splash, Home, Map Select, Level Select, and Game Over are pure CSS + HTML — reimagined with neon glassmorphic panels, perspective grid floors, and scanline overlays. No Three.js needed for menus.

Map themes

A JS theme object maps each of the 9 maps to a primary + secondary neon color. On map selection, CSS variables + Three.js material colors update instantly. No page reload.
